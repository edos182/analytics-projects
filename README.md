# Data Analytics Portfolio

Портфолио учебных проектов по анализу данных, SQL и визуализации данных.

## Проекты

### 1. Секреты Тёмнолесья

Анализ игрового поведения пользователей MMORPG.

В рамках проекта:
- анализ доли платящих игроков;
- исследование внутриигровых покупок;
- выявление аномальных транзакций;
- анализ популярных игровых предметов;
- исследование влияния расы персонажа на покупательскую активность.

Навыки и инструменты: PostgreSQL, SQL, анализ данных, агрегирование данных, статистический анализ, анализ пользовательского поведения, подготовка бизнес-выводов и рекомендаций.

📂 Папка проекта: dark_forest_project

---

### 2. Анализ рынка недвижимости

Анализ рынка жилой недвижимости Санкт-Петербурга и Ленинградской области.

В рамках проекта:
- исследование времени активности объявлений;
- анализ факторов, влияющих на скорость продажи недвижимости;
- исследование сезонности рынка;
- сравнение показателей Санкт-Петербурга и Ленинградской области;
- подготовка практических рекомендаций для бизнеса.

Навыки и инструменты: PostgreSQL, SQL, анализ данных, работа с датами, очистка данных и обработка выбросов, сегментация данных, подготовка бизнес-выводов и рекомендаций.

📂 Папка проекта: real_estate_market_analysis

---

### 3. Анализ конференций TED

Анализ данных конференций TED и разработка интерактивного дашборда в DataLens.

В рамках проекта:
- анализ конференций TED и их структуры;
- исследование популярных тем и тегов выступлений;
- анализ просмотров, аплодисментов и реакции аудитории;
- исследование характеристик успешных спикеров;
- разработка интерактивного дашборда для поддержки принятия решений.

Навыки и инструменты: SQL, PostgreSQL, DataLens, анализ данных, агрегирование данных, визуализация данных, разработка интерактивных дашбордов.

📂 Папка проекта: ted_conference_dashboard
